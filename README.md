IPL Data Analysis Dashboard
Project Overview

This project analyzes Indian Premier League (IPL) match data to uncover insights about team performance, player statistics, venue scoring patterns, and match trends. The analysis was performed using SQL, Python, and Power BI to build an interactive analytics dashboard.

Tools Used

SQL (MySQL) – Data cleaning and querying

Python (Pandas, Matplotlib) – Exploratory data analysis

Power BI – Interactive dashboard development

Dataset

The dataset contains ball-by-ball IPL match data including:

Match details

Player performances

Runs and wickets

Venue information

Match results

Project Workflow

Raw IPL datasets were imported into SQL database.

SQL queries and views were created to calculate player statistics and match summaries.

Python was used for exploratory data analysis and initial visualizations.

The cleaned data was connected to Power BI.

Interactive dashboards were built to analyze batting, bowling, and team performance.

Dashboard Features

The Power BI dashboard contains four pages:

1 Overview Dashboard

Total matches KPI

Total wickets KPI

Toss impact percentage

Average match score

Top teams by total wins

Season trend analysis

2 Batting Analysis

Top 10 run scorers

Top strike rate batters

Runs distribution by venue

Runs trend by season

3 Bowling Analysis

Top 10 wicket takers

Economy rate comparison

Wickets by season

Wickets by venue

4 Key Insights

Most successful teams

Top performing batters

Leading wicket takers

Venue scoring patterns

Toss impact on match results

Key Insights

Mumbai Indians are the most successful team with the highest match wins.

Virat Kohli leads the all-time IPL run scoring list.

Yuzvendra Chahal is the leading wicket-taker.

Certain venues consistently produce higher scoring matches.

Toss impact on match results is relatively minimal.
